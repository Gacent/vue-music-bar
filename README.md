## Knowledge
1. vue-cli3
2. vue-router
3. vuex
4. audio事件
5. 时间，秒转时filter
6. Detail组件是单个播放组件，有效语音播放为单个，传入list（可选，如果不填则走api请求获取list），传入index为指定播放的下标
